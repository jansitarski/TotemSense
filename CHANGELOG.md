# Changelog

## [v3] - 2026-05-20

- Renamed project from CCPlates to TotemSense.
- Added Hellfire Warders panel to README collage.

## [v2] - 2026-05-16

- Added Roadmap section to README (Poison/Disease/Grounding totem tracking, standalone addon).
- Added missing fear mobs: Gronn-Priest, Vashj'ir Honor Guard, Illidari Nightlord, Darkheart.

## [v1] - 2026-05-08

- Initial release.
- Plater Nameplates mod for WoW TBC Classic Shamans: shows a Tremor Totem icon on nameplates of NPCs that cast Fear, Charm, or Sleep.
- NPC database covering TBC raids and dungeons, Classic raids and dungeons.
- Icon scales relative to health bar height; configurable size, position, and texture.
- Wago.io packaging and GitHub Release workflow for the Plater import string.
